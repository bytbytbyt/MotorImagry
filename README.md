# Motor Imagry
## Introduction
A motor Imagry task based on EEGNET and EEGTCNET. 
## Environment
* Ubuntu 22.04.5 LTS
* GPU NVIDIA GeForce RTX 4060
* CUDA 11.3.1
* cudnn 8.2.1.32
* python 3.7
* TensorFlow 2.7
* matplotlib 3.5
* NumPy 1.20
* scikit-learn 1.0
* SciPy 1.7
## Dataset
The dataset is BCI VI-4a http://bnci-horizon-2020.eu/database/data-sets.
